# Home Expense Tracker (Java Swing + MySQL)

A desktop-based **Home Expense Tracker** application built using **Java Swing**, **JDBC**, and **MySQL**.  
It allows users to log in, add daily expenses, view expense history, and generate monthly/category-wise reports.
